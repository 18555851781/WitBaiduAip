WitBaiduAip
===
Unity3d中的百度语音识别和语音合成Restful接口的封装，适用于全平台

本工程基于Unity 2017.3.0f3 (Scripting Runtime Version: .Net 3.5)

工程中包含了2个Demo场景
- 语音识别，见Assets\WitBaiduAip\Examples\Asr.unity
- 语音合成，见Assets\WitBaiduAip\Examples\Tts.unity
